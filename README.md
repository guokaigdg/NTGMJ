## 📦 安装

```
npm install -g ddkit

//or

yarn global add ddkit
```

## 创建项目

```
ddkit create my-app
```

## 命令

```
ddkit -v 查询版本号
ddkit --help 获取帮助
ddkit update 检查更新
```

## 本地开发

```
npm link

之后正常使用ddkit 命令
```

## 发布npm
```
npm publish 
```

## 工具

-   commander （Node 的命令行解析库）
-   inquirer （实现命令行之间的交互）
-   fs-extra （基础文件操作库）
-   chalk （粉笔:控制台字符颜色样式）
-   ora （优雅的进度条）
-   download （实现文件远程下载）
-   handlebars （实现模板字符替换）
-   log-symbols （为各种日志级别提供着色符号）
-   update-notifier （npm 在线检查更新）
